My learning journey into web development



Week 1

This week was where i started my web development journey. I had no idea what was web development before hand and my only experience with it was with wix.com to create a portfolio for my DSA.



Wix logo (https://commons.wikimedia.org/wiki/File:Wix.com_Logo.png)

Through the lessons, I learnt what was web development and the diffrent attributes with it such as the diffrence between a front-end and a back-end developer. I also learnt the basics of html and the most useful tags we can utilize. My lecturer also taught us some css so that we could create a table with borders. After these lessons, i fully understand what a web developer does and how to utilize html.



HTML 5 logo (https://upload.wikimedia.org/wikipedia/commons/thumb/6/61/HTML5_logo_and_wordmark.svg/180px-HTML5_logo_and_wordmark.svg.png)

Some HTML tags i have learnt

Tags Utilization
< > Tag Indicator
html and !DOCTYPE Indicate it is a html document
head Head or top of the document
body Middle of the document
p Paragraph
h1-6 Heading font for the text
img src="" Image
alt= "" Alternate text to diplay if image didnt load(Used in img)
a href="" Hyper link
target="_blank" Open hyper link in new page(Used in a)
br Spacing
hr Creates a line
table Creates a table
tr Table row
td Table details




Week 2

After learning html last week, we were then taught how to use sematic html this week to organise our html code such that we can identify sections easier. We learnt tags such as:

  • header
  • footer
  • section
  • nav
  • article
  • aside
and more.



Sematic html diagram (https://www.w3schools.com/html/html5_semantic_elements.asp)

We were then taught the basics of css and how to set up css into a website, color text, use fonts, aglin text, create #special ids and .classes and use sematic html <> to specify styles to diffrent text and sections.
